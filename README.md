# ☕ Brew & Bean Café — Website

A beautifully designed, fully responsive website for **Brew & Bean Café**, a specialty coffee shop and artisan food destination located in Bengaluru, India. Built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

---

## ✨ Features

| Feature | Description |
|---|---|
| **Hero Section** | Fullscreen hero with animated badge, CTA buttons, and smooth scroll indicator |
| **About Section** | Brand story with image, badge overlay, and highlight features (Farm-to-Cup, Chef-Crafted, Cozy, Eco-Friendly) |
| **Live Stats** | Animated counters for years, coffees served, customers, and menu specials |
| **Tabbed Menu** | Interactive menu with four tabs — Coffee, Food, Pastries, and Specials — each with cards, pricing, and tags |
| **Photo Gallery** | Masonry-style gallery showcasing the café interior, brews, kitchen, and bakery |
| **Testimonials** | Carousel slider with customer reviews, star ratings, dot navigation, and auto-play |
| **Table Reservation** | Fully functional reservation form with date/time picker, guest count, occasion selector, and success state |
| **Contact Form** | Contact section with message form, embedded Google Maps, and info cards |
| **Newsletter** | Email subscription form in the footer |
| **Back to Top** | Scroll-triggered floating button for quick navigation |
| **Responsive Navbar** | Sticky navigation with mobile hamburger menu and smooth overlay transition |

---

## 💼 Business Pitch: How This Website Drives Growth

**The Challenge:** Local cafés often struggle with high commission fees from third-party delivery apps, lack of a centralized online presence to capture local search traffic, and manual reservation management.

**The Solution:** This custom website acts as a premium, 24/7 digital storefront for Brew & Bean Café. 
- **Direct Reservations:** The integrated online reservation system reduces phone call volume and staff overhead, capturing bookings directly without third-party fees.
- **Visual Menu Appeal:** A high-quality digital menu with a responsive gallery entices customers, up-selling premium items before they even arrive.
- **Brand Trust & SEO:** Customer testimonials build instant credibility, while a fast, mobile-optimized and SEO-friendly structure ensures the café ranks higher in local searches (e.g., "coffee shops near me").
- **Customer Retention:** The built-in newsletter signup allows the café to build an owned audience for promotions, driving repeat visits. 

**The Result:** A modern digital presence that elevates the brand, reduces reliance on expensive aggregator platforms, and smoothly converts online browsers into loyal, in-store patrons.
